# email-ai-assistant

AI-powered email reply suggestions using a local LLM (Ollama) and a FastAPI backend.

## 🎯 Purpose

This project explores how AI can assist with writing quick, context-aware email replies while keeping all data local.

It is inspired by my [`sms-ai-assistant`](https://github.com/jbakchr/sms-ai-assistant) project, but focused on email workflows.

## 🚧 Initial scope (MVP)

The first version will:

- Accept email text as input
- Generate 1–3 short reply suggestions
- Use a local LLM via Ollama
- Be triggered manually (no Gmail integration yet)

## 🛠️ Planned tech stack

- Python
- FastAPI
- Ollama (local LLM)
